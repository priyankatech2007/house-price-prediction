# 🏠 House Price Prediction using Machine Learning

## 📌 Problem Statement
The objective of this project is to predict house prices using various housing features from the Kaggle dataset.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## 📊 Model Used
- Linear Regression

## 📈 Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## 🚀 Results
The model achieved strong predictive performance after feature engineering and scaling.

## 📂 Dataset
Kaggle - House Prices: Advanced Regression Techniques
